KeyBinder class is for the creating UI Icons for gamepad buttons when the game client has a connected gamepad or the game client is runned on a game console.
When you create a different UI buttons, you should use the KeyBinder to not broken the cross platform key controller.
If you won't use the gamepad or gameconsole, then you don't need the KeyBinders.
